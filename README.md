## How to
Create and fill **.env** as **.env.example**

Start mongo in docker container
```sh
docker-compose up -d --build
```

Move to back-end dir
```sh
cd back-end
``` 

Create and fill **.env** as **.env.example**

Start back-end
```
npm i && npm run build && npm run start
```
